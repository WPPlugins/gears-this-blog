== Description ==

Gears This Blog allow your blog's visitors to store the blog's content on their computer for offline navigation, 
this, with the help of Google Gears.

== Installation ==

Installation guide : 

1. Create `gears-this-blog` directory in `/wp-content/plugins/`
2. Upload all files to the `/wp-content/plugins/gears-this-blog/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Enjoy it !
